Java Variables

Variables are containers used to store data values. In Java, every variable must have a data type. This tells the program what kind of data the variable will hold.

Example 
int age = 18;
String name = "Janella";
System.out.println(name + " is " + age);

Variables make programs flexible and reusable by allowing data to change during execution.

Java Data Types

Java is a statically typed language, which means variables must be declared with a data type. Data types specify the size and type of values that can be stored.

Example 
int number = 100;
boolean isActive = true;
double price = 99.99;

Common data types include int, double, char, boolean, and String.

Java Arrays

Arrays are used to store multiple values in a single variable. All elements in an array must be of the same data type.

Example 
int[] numbers = { 10, 20, 30 };

System.out.println(numbers[0]);
System.out.println(numbers.length);

Array indexes start at zero. Arrays are commonly used with loops to process data efficiently.

Java Loops

Loops are used to execute a block of code repeatedly as long as a condition is true. Java provides several types of loops.

Example 
for (int i = 0; i < 5; i++) {
    System.out.println(i);
}

Common loops include for, while, and do-while.

Java Methods

Methods are blocks of code that perform a specific task. They help reduce code repetition and improve program structure.

Example 
static void greet() {
    System.out.println("Hello World");
}

Methods can accept parameters and return values.

Java Classes and Objects

Java is an object-oriented programming language. Classes define objects, and objects are instances of classes.

Example 
class Student {
    int id;
}

Student s = new Student();
s.id = 1;

Objects allow programs to model real-world entities.

Java Abstract Classes

An abstract class cannot be used to create objects. It is designed to be inherited by other classes.

Example 
abstract class Animal {
    abstract void sound();
}

class Dog extends Animal {
    void sound() {
        System.out.println("Bark");
    }
}

Abstract classes help enforce rules and structure in large programs.
