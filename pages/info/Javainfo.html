Java Variables

Variables are containers used to store data values. In Java, every variable must have a data type. This tells the program what kind of data the variable will hold.

Example 
int age = 18;
String name = "Janella";
System.out.println(name + " is " + age);

Variables make programs flexible and reusable by allowing data to change during execution.

Java Data Types

Java is a statically typed language, which means variables must be declared with a data type. Data types specify the size and type of values that can be stored.

Example 
int number = 100;
boolean isActive = true;
double price = 99.99;

Common data types include int, double, char, boolean, and String.

Java Arrays

Arrays are used to store multiple values in a single variable. All elements in an array must be of the same data type.

Example 
int[] numbers = { 10, 20, 30 };

System.out.println(numbers[0]);
System.out.println(numbers.length);

Array indexes start at zero. Arrays are commonly used with loops to process data efficiently.

Java Loops

Loops are used to execute a block of code repeatedly as long as a condition is true. Java provides several types of loops.

Example 
for (int i = 0; i < 5; i++) {
    System.out.println(i);
}

Common loops include for, while, and do-while.

Java Methods

Methods are blocks of code that perform a specific task. They help reduce code repetition and improve program structure.

Example 
static void greet() {
    System.out.println("Hello World");
}

Methods can accept parameters and return values.

Java Classes and Objects

Java is an object-oriented programming language. Classes define objects, and objects are instances of classes.

Example 
class Student {
    int id;
}

Student s = new Student();
s.id = 1;

Objects allow programs to model real-world entities.

Java Operators

Operators are used to perform operations on variables and values. Java has several types of operators including arithmetic, assignment, comparison, and logical operators.

Example 
int x = 10;
int y = 5;

System.out.println(x + y);  // Addition: 15
System.out.println(x - y);  // Subtraction: 5
System.out.println(x * y);  // Multiplication: 50
System.out.println(x / y);  // Division: 2
System.out.println(x % y);  // Modulus: 0

Operators are essential for performing calculations and comparisons in your code.

Java Strings

Strings are used for storing text. A String variable contains a collection of characters surrounded by double quotes.

Example 
String greeting = "Hello World";
System.out.println(greeting);

String name = "John";
String age = "25";
System.out.println(name + " is " + age);

Strings have many useful methods like length(), toUpperCase(), toLowerCase(), and indexOf().

Java Conditionals (if...else)

Conditional statements are used to perform different actions based on different conditions. Use if to specify a block of code to be executed if a condition is true.

Example 
int age = 18;

if (age >= 18) {
    System.out.println("You are an adult");
} else {
    System.out.println("You are a minor");
}

You can also use else if to specify multiple conditions.

Java Inheritance

Inheritance allows a class to inherit properties and methods from another class. The class that inherits is called a subclass, and the class being inherited from is called a superclass.

Example 
class Animal {
    void eat() {
        System.out.println("Eating...");
    }
}

class Dog extends Animal {
    void bark() {
        System.out.println("Woof!");
    }
}

Inheritance promotes code reusability and establishes a relationship between classes.

Java Polymorphism

Polymorphism means "many forms". It occurs when we have many classes that are related to each other by inheritance. It allows you to write code that can work with objects of multiple types.

Example 
class Animal {
    void sound() {
        System.out.println("Some sound");
    }
}

class Dog extends Animal {
    void sound() {
        System.out.println("Woof");
    }
}

class Cat extends Animal {
    void sound() {
        System.out.println("Meow");
    }
}

Polymorphism allows the same method to behave differently for different objects.

Java Encapsulation

Encapsulation is the bundling of data (variables) and methods into a single unit called a class. It is one of the four fundamental OOP concepts. The idea is to hide the internal details and only expose what is necessary.

Example 
public class Person {
    private String name;
    
    public String getName() {
        return name;
    }
    
    public void setName(String name) {
        this.name = name;
    }
}

Use private to restrict access and public getter/setter methods to control access to data.

Java Abstract Classes

An abstract class cannot be used to create objects. It is designed to be inherited by other classes. Abstract classes can contain abstract methods (methods without a body) and regular methods.

Example 
abstract class Animal {
    abstract void sound();
}

class Dog extends Animal {
    void sound() {
        System.out.println("Bark");
    }
}

Abstract classes help enforce rules and structure in large programs.

Java Interfaces

An interface is a completely abstract class that is used to group related methods with empty bodies. Interfaces specify what a class should do, but not how it should do it.

Example 
interface Animal {
    void eat();
    void sleep();
}

class Dog implements Animal {
    public void eat() {
        System.out.println("Dog eats");
    }
    
    public void sleep() {
        System.out.println("Dog sleeps");
    }
}

A class can implement multiple interfaces, allowing for flexible code design.

Java Exception Handling

Exception handling is used to handle errors gracefully. When an error occurs, Java throws an exception. You can catch and handle exceptions using try-catch blocks.

Example 
try {
    int result = 10 / 0;
} catch (ArithmeticException e) {
    System.out.println("Cannot divide by zero!");
} finally {
    System.out.println("This always executes");
}

The finally block always executes, whether an exception occurs or not.
